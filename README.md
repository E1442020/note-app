# Note App

This project is a note-taking application built using React and TypeScript. It allows users to create, view, edit, and delete notes.

## Usage

Once the development server is running, you can access the note-taking application via your web browser. The application includes the following main functionalities:

Create Note: Add new notes with a title and content.
View Notes: View a list of all notes.
Edit Note: Modify the content or title of an existing note.
Delete Note: Remove notes from the list.
Example Usage
Open the application at http://localhost:3000.
Create a new note by clicking the "Add Note" button and filling out the form.
View the list of notes on the home page.
Edit a note by clicking the "Edit" button next to a note.
Delete a note by clicking the "Delete" button next to a note.

## Features

User-friendly Interface: Simple and intuitive interface for managing notes.
Type Safety: Built with TypeScript for better type safety and developer experience.
Real-time Updates: See changes immediately as you add, edit, or delete notes.
Responsive Design: Works on desktops, tablets, and mobile devices.

## Technologies

This project is built using the following technologies:

React: A JavaScript library for building user interfaces.
TypeScript (TSX): A typed superset of JavaScript that compiles to plain JavaScript.
CSS: For styling the application.
HTML5: Structure the web application.
